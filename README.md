# Skin-Cancer-Detection-using-Deep-Learning

Skin cancer is a common form of cancer caused mainly by excessive explosure of (UV) radiation, which damages skin cells. It can lead to Mutations that cause growth of abnormal cells, it being more dangerous Early detection of skin cancer based on different skin conditions is more important. To address this, the machine Learning models are trained to predict the skin cancer at early stage but the ML models are trained on limited data which results to inaccurate results. In proposed we have consider the dataset that contains 25k images, trained Convolution neural network at different convolution layers, epochs and model is validated. This Predictive model helps to detect skin cancer at early stage and to treat cancer at early which can save the lives.

This project leverages Deep Learning, specifically Convolutional Neural Networks (CNNs), to detect skin cancer at an early stage using image data.

## 📖 Overview
Traditional machine learning models often suffer from limited datasets, leading to lower accuracy. To overcome this, our project uses a dataset of 25,000+ skin images and applies CNN models with varying:

- Convolution layers.
- Epochs.
- Hyperparameters.

The model is trained and validated to improve prediction accuracy and reliability.

## 🚀 Features
- 🔍 Early detection of skin cancer.
- 🧠 Deep learning-based classification (CNN).
- 📊 Trained on large dataset (25K+ images).
- ⚡ Improved accuracy over traditional ML models.
- 🩺 Supports medical decision assistance.

## Model Architecture
- Convolutional Neural Network (CNN)
- Multiple convolution + pooling layers
- Activation: ReLU
- Fully connected dense layers
- Output layer for classification

## 📊 Dataset
- 📁 Total Images: ~25,000
- 🏷️ Categories: Different types of skin conditions
- 📌 Source: https://challenge.isic-archive.com/data/#2019

## 📈 Results
- ✅ Improved accuracy with deep learning
- 📊 Better generalization with large dataset
- 🧪 Validated using test dataset
